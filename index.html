Welcome

Hi there! Thanks for visiting our website. We would like to show you our program, CMAN. Written in 2 editions (Python and Java), we hope to make it cross-platform, so everyone can use it! Keep scrolling to get through the Q/A!

Q. What is CMAN? What does it stand for?

A. CMAN stands for Comprehensive Minecraft Archive Network. We hope to make a simple client to install mods without any trouble.

Q. How does CMAN work?

A. It works through JSON files containing information about the mods like the name, the version, the description, wheather it's an installer and the link. This is updated as it is downloaded everytime there is a new archive version (Archive being the JSON).
