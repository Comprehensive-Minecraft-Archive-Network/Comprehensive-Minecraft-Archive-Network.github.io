CMAN

Comprehensive Minecraft Archive Network

Welcome to the home of CMAN, The Comprehensive Minecraft Archive Network.

We're a group of Minecrafters who want to make mod installing a bit easier, so we made CMAN. CMAN is a modification (mod) release platform that is Work In Progress, so don't expect the first release to be this month, or possibly this year! To contribute to the project, please click here.
